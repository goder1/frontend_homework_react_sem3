Базовый URL

https://api.gamecatalog.com/v1

Аутентификация

Регистрация пользователя

POST /auth/register

Создание нового аккаунта пользователя

Вход в аккаунт

POST /auth/login

Аутентификация пользователя и получение токена

Выход из аккаунта

POST /auth/logout

Завершение сессии пользователя

Взаимодействие с играми

Поставить/убрать лайк

POST /games/{id}/favorite

Добавить или удалить игру из избранного

Добавить/убрать из желаемого

POST /games/{id}/wishlist

Добавить или удалить игру из списка желаемого

Получить список игр

GET /games

Получить каталог игр с фильтрацией

Поиск игр

GET /games/search

Поиск игр по названию и параметрам

Пользовательские данные

Получить профиль

GET /users/profile

Получить данные текущего пользователя

Получить ДНК-профиль

GET /users/dna-profile

Получить игровой ДНК-профиль пользователя

Добавить игру в коллекцию

POST /users/games

Добавить информацию о пройденной игре
