# GameCatalog

Приложение-каталог для поиска видеоигр и аналитики игровой активности пользователя. Используя его, можно найти игры, в которые не играл пользователь, а также детально проанализировать игровые предпочтения игрока

# Основной сценарий
### Пользователь заходит на сайт, смотрит на каталог игр. Далее переходит в избранное/профиль, его редеректит на страницу регистрации/авторизации. Он регистрирует аккаунт и входит в него. Далее пользователь заходит в свой аккаунт и добавляет в профиль несколько игр, в которые он играл. В конце пользователь может увидеть статистику по своей игровой активности в профиле

# Структура приложения
директория supabase - конфиги и миграции бд. src - директория со всеми файлами приложения. src/components - визуальные компоненты. src/lib - клиент бд. src/pages - страницы приложения. src/services - сервисная часть страниц.
src/store/slices - слайсы. src/styles - общие css. src/types - общие интерфейсы. src/App.tsx - точка входа с роутингом приложения
